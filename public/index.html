Descripción del proyecto

Se solicitó implementar un servidor web en Node.js con acceso a una base de datos PostgreSQL mediante consultas parametrizadas con la librería pg. El sistema expone cuatro endpoints REST que permiten crear, consultar, modificar y eliminar clientes desde una tabla con campos rut, nombre y edad.

El backend fue desarrollado con Express aplicando validaciones de entrada, manejo de errores y códigos HTTP correctos. Los endpoints fueron probados individualmente con Thunder Client antes de conectar el frontend. La interfaz web consume los servicios mediante fetch API y muestra los resultados en tiempo real.
